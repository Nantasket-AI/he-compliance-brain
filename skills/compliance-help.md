# Compliance Brain — Help

All available skills in the HE Compliance Brain plugin.

## Second Brain — Capture

| Command | Purpose |
|---|---|
| `/brain-capture` | Universal capture — routes to the right type automatically |
| `/brain-capture-meeting` | Structured meeting notes with action items |
| `/brain-capture-reading` | Reading notes, highlights, and takeaways |
| `/brain-capture-decision` | Decision log with rationale and alternatives |
| `/brain-capture-person` | Notes on a person or key contact |

## Second Brain — Ingest from M365

| Command | Purpose |
|---|---|
| `/brain-ingest-email` | Capture key emails or threads from Outlook |
| `/brain-ingest-teams` | Capture a Teams conversation or meeting transcript |
| `/brain-ingest-sharepoint` | Pull a document from SharePoint into the brain |
| `/brain-ingest-calendar` | Capture context from a calendar event or meeting invite |

## Second Brain — Retrieve & Synthesize

| Command | Purpose |
|---|---|
| `/brain-retrieve` | Search notes by topic or keyword |
| `/brain-synthesize` | Synthesize all notes on a topic into a structured document |
| `/brain-daily` | Daily review: recent captures, open action items, focus |

## Compliance Workflows

| Command | Purpose |
|---|---|
| `/compliance-gap` | Gap analysis against a regulatory requirement |
| `/compliance-risk` | Add or update a risk register entry |
| `/compliance-audit-prep` | Prepare evidence and talking points for an audit |
| `/compliance-policy` | Draft or revise a policy or procedure |
| `/reg-monitor` | Summarize recent regulatory changes in a domain |
| `/reg-map` | Map a regulation to internal controls and owners |

Use `/compliance-help` any time to see this list.
