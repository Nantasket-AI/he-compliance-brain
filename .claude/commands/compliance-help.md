# Compliance Brain — Help

List all available compliance brain skills and their purpose.

## Skills: Second Brain / PKM

| Command | Purpose |
|---|---|
| `/brain-capture` | Capture a note, insight, or reference into the knowledge base |
| `/brain-retrieve` | Search and surface relevant knowledge on a topic |
| `/brain-synthesize` | Synthesize notes on a topic into a structured summary |

## Skills: Compliance Workflows

| Command | Purpose |
|---|---|
| `/compliance-gap` | Run a gap analysis against a regulatory requirement |
| `/compliance-risk` | Add or update an entry in the risk register |
| `/compliance-audit-prep` | Prepare evidence and talking points for an upcoming audit |
| `/compliance-policy` | Draft or revise a policy or procedure document |
| `/reg-monitor` | Summarize recent regulatory changes in a domain |
| `/reg-map` | Map a regulation to internal controls and owners |

Use `/compliance-help` any time to see this list.
